# Windows Active Directory Home Lab

## Status

✅ Completed

## Overview

This project simulates a small corporate Active Directory environment built on Windows Server 2022.

## Technologies

- Windows Server 2022
- Active Directory Domain Services
- DNS
- Group Policy
- PowerShell
- Oracle VirtualBox

## Implemented Features

### Domain Controller

- Installed Windows Server 2022
- Promoted server to Domain Controller
- Created forest `homelab.local`
